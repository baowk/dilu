# Dilu v1.2.2 Release Notes

发布日期：2026-03-16  
版本标签：`v1.2.2`

## 本次重点

1. 目录结构更贴近 Go 规范  
   - 业务模块统一为 `internal/modules/<module>`  
   - 公共组件统一在 `internal/common`

2. 生成器模板与持久层路径对齐  
   - GORM-Gen 生成路径与模板引用路径一致  
   - 生成代码更易维护，避免路径不一致导致编译失败

3. 文档结构清晰化  
   - `docs/` 用于用户与接口文档（含 swagger）  
   - `dev-docs/` 用于架构/规范/AI 开发过程文档

4. 版本输出改为构建注入  
   - `dilu version` 默认显示 `dev`  
   - 发布时用 `-ldflags` 注入版本号

## 兼容性说明
- 目录结构调整后，原有 `internal/sys` / `common` 目录需要迁移。  
- 若你自定义模板或路径，请同步调整引用路径。

## 构建示例
```bash
go build -ldflags "-X 'dilu/cmd/version.Version=v1.2.2'" -o dilu
```
